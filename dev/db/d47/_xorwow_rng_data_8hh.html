Resize and initialize with the seed stored in params.

Resize and seed the RNG states.

cuRAND and hipRAND implement functions that permute the original (but seemingly arbitrary) seeds given in Marsaglia with 64 bits of seed data. We simply generate pseudorandom, independent starting states for all data in all threads using MT19937.
