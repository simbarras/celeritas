template<template< class > class T>
struct celeritas::detail::StaticSurfaceAction< T >

Convert a surface type to a class property via a traits class.

The traits class T must be templated on surface type, and (like std::integral_constant ) have a 

     constexpr value_type operator()() const noexcept
*

member function for extracting the desired value.
