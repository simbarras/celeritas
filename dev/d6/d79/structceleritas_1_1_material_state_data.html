Detailed Description

template<Ownership W, MemSpace M>
struct celeritas::MaterialStateData< W, M >

Store dynamic states of multiple physical particles.

The size of the view will be the size of the vector of tracks. Each particle track state corresponds to the thread ID (ThreadId).

The "element scratch space" is a 2D array of reals, indexed with [track_id][el_component_id], where the fast-moving dimension has the greatest number of element components of any material in the problem. This can be used for the physics to calculate microscopic cross sections.
