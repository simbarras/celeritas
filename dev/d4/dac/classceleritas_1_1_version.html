Detailed Description

Simple version comparison.

This is intended to be constructed with version numbers from celeritas_cmake_strings.h and used in unit tests. It can be used in if constexpr expressions with preprocessor macros. In the constructor documentation, x/y/z correspond to major/minor/patch.

Version(4) == Version(4.0) == Version(4.0.0)
Version(3.1) > Version(3)
